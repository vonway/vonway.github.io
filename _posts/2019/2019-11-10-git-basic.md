# git基础操作

## git configure
```
$ git config --global user.name "Your Name"
$ git config --global user.email "your.email@gmail.com"
$ git config --global core.editor <editor>    # <editor>: vi, vim, etc.

// 生成ssh的密钥
$ mkdir -p $HOME/.ssh
$ chmod 700 $HOME/.ssh
$ ssh-keygen -t rsa 
// 添加到gerrit中

```

## clone操作
```
git clone url
```
